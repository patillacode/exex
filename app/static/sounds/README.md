# Sound Files for Expresión Exprés

This directory should contain audio files for the game. You need to add the following sound files:

## Required Sound Files

1. `beep.mp3` - A short beep sound for the timer's regular notification
2. `buzzer.mp3` - A more prominent buzzer sound for when the timer runs out

## Sound File Requirements

- Files should be in MP3 format for best browser compatibility
- Keep file sizes small (under 100KB each) for performance
- Beep sound should be short (less than 0.5 seconds)
- Buzzer sound should be distinct and attention-grabbing (1-2 seconds)

## Sources for Sound Files

You can obtain suitable sound files from:
- Free sound libraries like [Freesound](https://freesound.org/)
- Public domain sound collections
- Create your own using audio tools

## Adding Sounds During Development

You can add these sounds during development by placing them directly in this directory. Make sure the filenames exactly match what's expected by the application.
